Activity 1

| ID  | Requirement                                                                          | Rationale                                                                                                                                                                                                                                               |
| --- | ------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| F1  | The system shall allow the user to enter a bus route                                 | Users need to be able to choose what route they will drive                                                                                                                                                                                              |
| F2  | The system must validate the bus route                                               | Bus drivers should not be able to enter bus routes that don't exist                                                                                                                                                                                     |
| F3  | The system shall provide stop approach alerts                                        | To inform users and ensure they dont miss the stops                                                                                                                                                                                                     |
| F4  | The system will automattically provide detours                                       | The system needs to be able to reroute the route in the case of road obstacles                                                                                                                                                                          |
| F5  | The system shall allow drivers to report road obstacles and delays                   | The system needs to be able to recieve real time information on locations of hazards and delays                                                                                                                                                         |
| F6  | The system shall provide turn by turn navigation instructions                        | Users will need navigation at every turn to ensure they remain on the correct route                                                                                                                                                                     |
| F7  | The user shall be able to confirm stop arrivals and trip events                      | Confirmation of events during the route is crucial for the accuracy of the real time data  the system relies on                                                                                                                                         |
| F8  | The system will be able to correct for wrong turns                                   | If the user makes a wrong turn, the system will reroute its instructions to navigate to the next stop from the bus' current position                                                                                                                    |
| F9  | The system shall provide alerts when the user is in an active school zone            | By providing alerts when in an active school zone, the system can ensure the user is fully informed, mitigating potential risks                                                                                                                         |
| F10 | The system shall provide information of how far ahead or behind schedule the user is | By providing clear information about how far ahead or behind schedule, as well as how close they are to other busses on the route, the system can allow the user to make informed decisions regarding their driving and how long they stay at each stop |
