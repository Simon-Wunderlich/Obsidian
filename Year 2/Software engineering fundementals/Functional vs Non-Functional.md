# Functional
- Services system should provide
- How the system reacts to particular inputs / in particular situations
- May state what the system should not do
## Functional user requirements
Describe in high level what the system should do
## Functional system requirements
Describe system services in detail
# Non-Functional
- Constrains on functions / services
	- Timing
	- Development process constraints
	- Standards
	- Resources
- Impacts whole system
