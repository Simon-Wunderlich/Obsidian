# Swap lowest value into lowest unsorted position

EXAMPLE
Smallest letter is A
**A**|XEMPLE
Next smallest letter is E
A**E**|**X**MPLE
Next smallest letter is E
AE**E**|MPL**X**
Next smallest letter is L
AEE**L**|P**M**X
Next smallest letter is M
AEEL**M**|PX
Next smallest letter is P
AEELM**P**|X
Next smallest letter is X
AEELMP**X**|
