$O(f(n))$ - Set of functions with a **same / lower** growth rate than f(n)
$\Omega(f(n))$ - Set of functions with a **same / higher** growth rate than f(n)
$\Theta(f(n))$ - Set of functions with the **same** growth rate as f(n)
$o(f(n))$ - Set of functions with a **lower** growth rate than f(n)
$\omega(f(n))$ - Set of functions with a **higher** growth rate than f(n)

When calculating time complexity, ignore negative signs

#### Comparing f(n) and g(n)

$f(n) \in O(g(n))$
