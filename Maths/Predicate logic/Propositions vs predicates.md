# Predicate
Function on a domain that returns a true of false value
##### Single variable
Property of x
$$
1+x=2
$$
##### Multi variable
Relation between x and y
$$
x+y=2
$$


## Bound vs Free variables
### Free variable
Can take any value in the domain
$$
f(x)
$$
### Bound variable
Is bound to a [[Quantifiers | quantifier]]
$$
\exists xP(x)
$$
